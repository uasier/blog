【实用技巧】在markdown 文档中添加视频的方法

代码块：

<iframe 
    width="800" 
    height="450" 
    src="//player.bilibili.com/player.html?aid=16688970&cid=27215876&page=1" 
    scrolling="no" 
    border="0" 
    frameborder="no" 
    framespacing="0" 
    allowfullscreen="true">
    暂时不支持视频插入，请联系我，谢谢。
</iframe>

注意事项：
    1. 必须要标明height 否则iframe就会特别小
    2. 注意属性是需要引号的，单引双引都ok
    3. 有些网站分享出来的代码块是有问题的，需要调整（for example 哔哩哔哩 需要调整为以上格式）
